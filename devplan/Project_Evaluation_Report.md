## 🎯 프로젝트 목표 및 비전

### 프로젝트 목적
**Vibe Coding Report**는 AI 기반 프로젝트 평가 및 개선 보고서 자동 생성 VS Code 확장입니다. 개발자가 AI 에이전트와 협업하여 프로젝트를 지속적으로 개선할 수 있도록 지원합니다.

### 핵심 목표
1. **원클릭 프로젝트 분석** - 버튼 하나로 프로젝트 전체를 스캔하고 분석 프롬프트 생성
2. **삼중 보고서 시스템** - 평가 보고서, 개선 보고서, AI 프롬프트 파일 분리 관리
3. **개선 추적** - 적용된 개선사항을 추적하여 중복 제안 방지
4. **AI 에이전트 연동** - 생성된 프롬프트를 Copilot Chat에 전달하여 즉시 적용
5. **다국어 지원** - 보고서는 한국어, AI 프롬프트는 영어로 분리 작성

### 대상 사용자
- Vibe Coding(AI 페어 프로그래밍)을 활용하는 개발자
- 프로젝트 품질을 지속적으로 모니터링하고 싶은 팀
- AI 기반 코드 리뷰 및 개선을 원하는 개발자

### 📊 분석 요약

**vibereport-extension v0.2.1**은 VS Code 내에서 AI 기반의 프로젝트 분석 및 보고서 생성을 자동화하는 정교한 **메타-도구(Meta-tool)**입니다.

**v0.2.0 → v0.2.1 주요 변경사항:**
